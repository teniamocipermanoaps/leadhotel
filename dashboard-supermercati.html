⚙ Configura sorgente Google Sheet

Incolla gli URL CSV pubblicati delle due tab del tuo Google Sheet "Lead Supermercati APS". Lasciali vuoti per modalità locale (dati solo nel tuo browser). Vedi SETUP.md per istruzioni.

Come ottengo gli URL pubblicati?
  1. Apri il Google Sheet.
  2. File → Condividi → Pubblica sul web.
  3. Nel selettore di sinistra: scegli la tab (es. "Supermercati"); a destra: Valori separati da virgole (.csv).
  4. Clicca Pubblica → copia l'URL → incollalo sopra.
  5. Ripeti per la tab "Eventi".
